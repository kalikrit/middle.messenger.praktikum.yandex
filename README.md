# Мессенджер
## Проект написан на javascript использует шаблонизатор handlebars
## version 1.3.4 / 16.11.2023

### проект использует:
 - node.js ^20
 - TS ^5.2.2
 - eslint-config-airbnb ^19.0.4
 - stylelint ^15.11.0

## Установка:
npm install

## Запуск
npm run start

## Билд
npm run build

## Запус сервера express
node ./server.js

## сверстанные страницы:
- Авторизация
- Регистрация
- Список чатов
- Настройки пользователя
- Страница 404
- Страница 500
